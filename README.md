# 🌾 FarmConnect: Direct Roots to Retail

**FarmConnect** is a revolutionary digital bridge designed to empower local farmers and wholesale buyers by removing middlemen and providing real-time market intelligence. Built for transparency, efficiency, and sustainability.

## 🚀 Key Features

### 🧑‍🌾 Farmer Empowerment
- **Market Intelligence Engine**: Real-time regional price guidance (Min/Max) ensures farmers never undersell.
- **Dynamic Listings**: Easy management of harvest categories, units, and inventory.
- **Direct Negotiation**: Peer-to-peer price bidding with buyers.

### 🏬 Wholesale Marketplace
- **Direct Sourcing**: High-quality produce directly from verified farmers.
- **Smart Negotiation**: Interactive bidding modal with market-data feedback to ensure fair pricing.
- **Order Tracking**: Seamless checkout with Home Delivery support.

### 🚚 Logistics Command Center
- **Supply Chain Map**: Interactive **Leaflet.js** map tracking 🚜 farm pickups and 🏬 consumer deliveries.
- **Real-time Status**: Live tracking of "Pending", "In Transit", and "Delivered" shipments.
- **Network Discovery**: Public directory for buyers to find verified local logistics partners.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System).
- **Core Logic**: JavaScript (ES6+ Mobile-first approach).
- **Backend & Auth**: Firebase Firestore (Real-time DB).
- **Interactive Mapping**: Leaflet.js (Voyager Tiles).
- **Design Assets**: FontAwesome 6, Google Fonts (Inter/Outfit).

## 🏁 Getting Started
1. Clone the repository: `git clone [Your Repo URL]`
2. Open `index.html` in any modern browser.
3. No build step required! Just valid Firebase credentials in `assets/firebase-config.js`.
